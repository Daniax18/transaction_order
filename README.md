The main objective of this system is to enable secure order transaction operations through video recognition, using RSA encryption to protect the data and ensure transaction integrity.

Architecture : Microservice using Event-Driven Architecture (EDA) with RabbitMQ
Tech for back : C# and Java
Tech for frontend: Next.js
BDD : Postgres / MongoDb
